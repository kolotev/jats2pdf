Overview

RESTful API around Antenna House Formatter application, which is used for converting manuscript XML tagged according to JATS to PDF.

Project layout

The following layout is available for those who have access to the source repository of this project.

    bin/    # scripts for running variuos tasks/modules.
    etc/    # settings.
    doc/    # documentation of this project.
    src/    # source code for this project.
    tests/  # variuos tests executed during building process.
    requirements/   # dependencies of this project
    doc/fsm_diagrams    # diagrams of finate state machines used in this project.
